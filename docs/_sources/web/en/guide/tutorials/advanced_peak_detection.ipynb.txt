    "# Peak Detection\n",
    "\n",
    "`gwexpy` provides the `find_peaks` method for easily detecting peaks in signals contained in `TimeSeries` or `FrequencySeries`. This internally wraps `scipy.signal.find_peaks`, but allows intuitive parameter specification using physical units (Hz, seconds, etc.).\n",
    "\n",
    "The `find_peaks` method returns the following two items:\n",
    "- **Peak Series**: A new Series object containing only the peak points\n",
    "- **Properties**: A dictionary of peak properties (such as prominence and width) returned by scipy"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "cell-1",
   "metadata": {},
   "outputs": [],
   "source": [
    "import matplotlib.pyplot as plt\n",
    "import numpy as np\n",
    "from astropy import units as u\n",
    "\n",
    "from gwexpy import FrequencySeries, TimeSeries"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "cell-2",
   "metadata": {},
   "source": [
    "## 1. Peak Detection in FrequencySeries"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "cell-3",
   "metadata": {},
   "outputs": [],
   "source": [
    "# Create mock data\n",
    "df = 0.5 * u.Hz\n",
    "f_axis = np.arange(0, 500, df.value) * u.Hz\n",
    "data = np.exp(-f_axis.value / 100) + np.random.normal(0, 0.05, len(f_axis))\n",
    "\n",
    "# Add peaks\n",
    "peak_freqs_true = [50, 60, 120, 125, 300] * u.Hz\n",
    "for f in peak_freqs_true:\n",
    "    idx = int(f.value / df.value)\n",
    "    data[idx] += 1.0\n",
    "\n",
    "spec = FrequencySeries(data, df=df, unit=\"V/Hz\")\n",
    "\n",
    "# --- Basic detection (threshold only) ---\n",
    "# peaks is returned as a FrequencySeries object\n",
    "peaks, _ = spec.find_peaks(threshold=0.5)\n",
    "\n",
    "plt.figure(figsize=(10, 5))\n",
    "plt.plot(spec, label=\"Spectrum\", alpha=0.7)\n",
    "plt.plot(peaks, \"x\", color=\"red\", label=\"Basic Peaks\")\n",
    "plt.title(\"Basic Peak Detection\")\n",
    "plt.xlabel(\"Frequency [Hz]\")\n",
    "plt.ylabel(\"Amplitude [V/Hz]\")\n",
    "plt.legend()\n",
    "plt.show()"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "cell-4",
   "metadata": {},
   "source": [
    "## 2. Constraints Using Physical Units (Distance & Width)"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "cell-5",
   "metadata": {},
   "outputs": [],
   "source": [
    "dist_constraint = 20 * u.Hz\n",
    "peaks_adv, props = spec.find_peaks(threshold=0.5, distance=dist_constraint)\n",
    "\n",
    "plt.figure(figsize=(10, 5))\n",
    "plt.plot(spec, label=\"Spectrum\", alpha=0.7)\n",
    "plt.plot(peaks_adv, \"o\", color=\"orange\", label=\"Distance Constraint (20Hz)\")\n",
    "plt.title(f\"Peak Detection with Distance Constraint ({dist_constraint})\")\n",
    "plt.xlabel(\"Frequency [Hz]\")\n",
    "plt.ylabel(\"Amplitude [V/Hz]\")\n",
    "plt.legend()\n",
    "plt.show()\n",
    "\n",
    "print(f\"Detected peaks: {peaks_adv.frequencies}\")"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "cell-6",

   "source": [
    "## 3. Peak Detection in TimeSeries"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "cell-7",
   "metadata": {},
   "outputs": [],
   "source": [
    "fs = 1024.0\n",
    "t = np.arange(0, 2.0, 1 / fs)\n",
    "ts_data = np.random.normal(0, 0.1, len(t))\n",
    "\n",
    "# Add pulses\n",
    "ts_data[int(0.5 * fs)] = 2.0\n",
    "ts_data[int(0.55 * fs)] = 1.8\n",
    "ts_data[int(1.2 * fs)] = 1.5\n",
    "\n",
    "ts = TimeSeries(ts_data, sample_rate=fs, unit=\"counts\")\n",
    "\n",
    "# Ignore consecutive pulses within 0.1 seconds\n",
    "peaks_t, _ = ts.find_peaks(height=0.5, distance=0.1 * u.s)\n",
    "\n",
    "plt.figure(figsize=(10, 5))\n",
    "plt.plot(ts)\n",
    "plt.plot(peaks_t, \"v\", color=\"red\", label=\"Detected pulses (>100ms apart)\")\n",
    "plt.title(\"Pulse Detection in TimeSeries\")\n",
    "plt.xlabel(\"Time [s]\")\n",
    "plt.ylabel(\"Raw Counts\")\n",
    "plt.legend()\n",
    "plt.show()"
