# ライセンス

## MIT ライセンス

GWexpy は **MIT ライセンス**の下で公開されています。これは、以下のことを許可する寛容なオープンソースライセンスです：

✓ 任意の目的（商用、学術、個人）での使用  
✓ ソフトウェアの変更と配布  
✓ ソフトウェアの再ライセンス  

**条件：**  
- ライセンスおよび著作権表示のコピーを含める
- ソフトウェアは「現状のまま」で提供され、保証はありません

## MIT が選ばれた理由

GWexpy が MIT ライセンスを選択した理由：

1. **採用を促進** - 寛容なライセンスが使用の障壁を軽減
2. **コラボレーションをサポート** - 派生作品は異なるライセンスを使用できる
3. **学術向け** - 研究ソフトウェアで広く使用されている
4. **商業向け** - 独自派生物の作成を許可

## サードパーティライセンス

GWexpy は以下のライセンスを持つオープンソースソフトウェアに依存しています：

| パッケージ | ライセンス |
|---------|---------|
| NumPy | BSD |
| SciPy | BSD |
| Pandas | BSD |
| Matplotlib | PSF |
| GWpy | GPL-3.0+ |
| Astropy | BSD |

完全な依存関係とそのライセンスのリストを確認するには、以下を実行してください：

```bash
pip show gwexpy
# または詳細情報：
pip-licenses
```
