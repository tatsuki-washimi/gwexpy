概念別ガイド

  • FFT の仕様と規約 (FFT_Conventions) ではフーリエ正規化、軸規約、対応 API を確認できます。

  • 前提条件と規約 では時刻系、FFT 規約、物理解釈の共通前提を確認できます。

  • スペクトル推定 (Spectral) では PSD、ASD、bootstrap 系推定の入口をまとめています。

  • フィッティング (gwexpy.fitting) では最小二乗、GLS、MCMC 系のフィッティング API をまとめています。

  • 検証済みアルゴリズム では、高度・理論向けの入口として、監査根拠付きの前提条件と API 対応を確認できます。

  • 数値的安定性と精度 では適応ホワイトニングなどの安定化方針を確認できます。

  • ノイズ生成 (Noise) では合成ノイズや代理波形生成の入口をまとめています。

橋渡しページ

  • 互換・補助 (Extra) では互換入口と追加 API を確認できます。

  • GWpy ユーザー向け移行ガイド では GWpy 移行ガイドを確認できます。

  • GWpy 差分 API 一覧 では GWpy 差分観点で API をたどれます。

  • Field API: ScalarField の基本 と ARIMA: 時系列予測 はリファレンスへ戻りやすい個別チュートリアル入口です。
