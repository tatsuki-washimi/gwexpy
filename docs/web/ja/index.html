GWexpy ドキュメントへようこそ

GWexpy は GWpy を拡張し、時系列および周波数系列データ解析のための新たなコンテナやユーティリティを提供します。

主な特徴:

  • TimeSeries/FrequencySeries 行列対応

  • 高次元フィールドAPI (ScalarField, VectorField, TensorField)

  • 数値計算の安定化機能

  • 実験データ解析向けユーティリティ

クイックスタート

まず インストール で環境構築し、次に クイックスタート で基本的な使い方を学びましょう。
