Getting Started

Welcome to GWexpy! This guide provides a learning path for getting started with GWexpy.

Prerequisites

  • Basic Python 3.9+ knowledge

  • NumPy fundamentals (array operations)

  • (Optional) GWpy experience

Learning Path

1. Installation

First, install GWexpy by following installation.

2. Quickstart

Learn the basics with quickstart.

3. Basic Data Structures (Recommended Order)

For Beginners

  1. intro_timeseries - Time series basics

  2. intro_frequencyseries - Frequency series basics

  3. intro_spectrogram - Spectrogram basics

  4. intro_plotting - Plotting features

For GWpy Users

  • Check the GWpy compatibility guide for migration information

4. Advanced Topics

Multi-channel & Matrix Containers

  • matrix_timeseries - Time series matrices

  • matrix_frequencyseries - Frequency series matrices

High-dimensional Fields (Field API)

  • field_scalar_intro - ScalarField introduction

  • scalarfield_slicing - Slicing guide (Important)

Advanced Signal Processing

  • advanced_fitting - Fitting methods

  • advanced_peak_detection - Peak detection

  • advanced_hht - Hilbert-Huang Transform

  • advanced_arima - ARIMA models

  • advanced_correlation - Correlation analysis
