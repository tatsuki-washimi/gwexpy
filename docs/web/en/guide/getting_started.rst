Getting Started
===============

Welcome to GWexpy! This guide provides a learning path for getting started with GWexpy.

Prerequisites
-------------

- Basic Python 3.9+ knowledge
- NumPy fundamentals (array operations)
- (Optional) GWpy experience

Learning Path
-------------

1. Installation
~~~~~~~~~~~~~~~

First, install GWexpy by following :doc:`installation`.

2. Quickstart
~~~~~~~~~~~~~

Learn the basics with :doc:`quickstart`.

3. Basic Data Structures (Recommended Order)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**For Beginners**

1. :doc:`tutorials/intro_timeseries` - Time series basics
2. :doc:`tutorials/intro_frequencyseries` - Frequency series basics
3. :doc:`tutorials/intro_spectrogram` - Spectrogram basics
4. :doc:`tutorials/intro_plotting` - Plotting features

**For GWpy Users**

- Check the GWpy compatibility guide for migration information

4. Advanced Topics

**Multi-channel & Matrix Containers**

- :doc:`tutorials/matrix_timeseries` - Time series matrices
- :doc:`tutorials/matrix_frequencyseries` - Frequency series matrices

**High-dimensional Fields (Field API)**

**Advanced Signal Processing**
