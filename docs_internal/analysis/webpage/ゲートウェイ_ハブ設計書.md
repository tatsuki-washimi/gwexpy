## 4. 実装上の注意事項

### Sphinx RTD テーマの制約
- `sphinx_rtd_theme` ではサイドバーを CSS で非表示にするのが最も確実（`.wy-nav-side { display: none !important }`）
- ゲートウェイ専用スタイルは `raw:: html` ブロック内に直接埋め込む、または gateway専用 CSS クラスで分岐する
- `|release|` と `|today|` は Sphinx が自動解決する（`conf.py` から取得）

### 英語版ハブ（`web/en/index.rst`）

- カード名・説明文は英訳し、リンク先は `gwexpy_for_gwpy_users_en` に切り替える

現行の4グループ（導入 / 学習 / 高度な機能 / リファレンス）は**変更しない**。
ページ本文の構成変更が主なスコープ。

### Phase 4: バージョン自動取得 + ビルド検証
**依存:** Phase 1, 2, 3 完了 | **優先度:** 🟢 低（最後の仕上げ）

#### 6-4-1. conf.py に release 変数を追加
- ファイル: `docs/conf.py`
- 追加内容:
  ```python
  from importlib.metadata import version as _get_version
  try:
      release = _get_version("gwexpy")
  except Exception:
      release = "dev"
  version = ".".join(release.split(".")[:2])  # Major.Minor
  ```
- これで `|release|` と `|version|` が自動解決される
