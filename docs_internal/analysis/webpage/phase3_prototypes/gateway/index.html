Scientific Python toolkit for gravitational-wave data analysis

GWexpy

Extended data containers, fitting workflows, and signal-processing utilities for researchers who need matrix-aware, unit-aware analysis without leaving Python.

GWpy を拡張し、行列・フィールド・フィッティング・信号処理を一つの解析導線にまとめる 科学計算向けツールキット。

cd gwexpy
pip install -e .

Three-line demo

from gwexpy.timeseries import FrequencySeriesMatrix
fsmtx = FrequencySeriesMatrix.read("data.hdf5")
fsmtx[2, 0].fit(model="lorentzian").plot()
