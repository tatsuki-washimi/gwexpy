How to Use

GWpy の拡張を、目的ベースでたどれる入口に整理する

このプロトタイプは、Gateway が「GWexpy とは何か」を伝えた後に、 利用者を Quickstart、チュートリアル、そして Reference に迷わず送るための ハブ骨格です。サイドバーは情報の階層を固定し、メインカラムは 読者別ルートとタスク別ルートを分離しています。

v0.x.x 想定 Python ≥ 3.9 Gateway からの遷移先
