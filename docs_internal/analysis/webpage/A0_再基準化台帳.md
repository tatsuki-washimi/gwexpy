## status 値定義

- limitation: `verify-changed-files --changed` currently has no docs quality-gate profile for this repo.
- `Implemented`: JA/EN の公開ソースと現在の docs 構成から見て、対象 finding は実装済みまたは公開面で充足済みと判断できる。
- `Needs Revalidation`: 一部は実装済みだが、公開 docs 上での見せ方・導線・再確認が必要。
- `Open`: 現時点では未充足で、公開 docs 側の追加修正が必要。
