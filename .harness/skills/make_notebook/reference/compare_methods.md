# Compare Methods in Notebooks

手法比較を notebook に組み込む時の specialized guidance。

## Procedure

1. 対象メソッドの実装差分をコード上で確認する
2. 数値的な違いと物理的な意味を分けて整理する
3. 同一入力で比較できるサンプルコードを用意する
4. 表と図を使って notebook section に落とし込む
5. 最後に「どの状況でどちらを選ぶか」を明記する

## Comparison Axes

- Algorithm
- Resolution / averaging
- Filtering / post-processing
- Physical interpretation
- Recommended use case
