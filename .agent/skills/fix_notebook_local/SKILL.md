# Fix Notebook (Local Only)

このスキルは、ライブラリ本体（`gwexpy/`）のコードには一切手を加えず、ノートブックのセル内容のみを修正してエラーを解決します。

## 手順

1.  **エラーの分析**:
    *   ノートブックのセルの記述ミス、インポート漏れ、データのパス指定ミスなどを特定します。

2.  **セルの修正**:
    *   ノートブックのコードセルの内容を書き換え、正しいAPIの使用法やパラメータに修正します。

3.  **安全性の担保**:
    *   ライブラリ側のバグであっても、このスキルではワークアラウンド（回避策）をノートブック側に記述することで対応します。

4.  **検証**:
    *   ノートブックを再実行し、エラーが解消されたことを確認します。
