# Available Models & Environments (as of 2026)

## Claude (Anthropic)

### Claude Opus 4.5 (Thinking)
- **Best for**: Complex physical/mathematical verification, difficult bug fixes, large-scale architectural changes
- **Strengths**: Top-tier analytical power, deep reasoning capabilities
- **Use Case**: Repository-wide analysis, physics validation, intricate design decisions

### Claude Sonnet 4.5
- **Best for**: General-purpose implementation, balanced reasoning and generation
- **Strengths**: Excellent all-round capability, good token efficiency
- **Use Case**: Most coding tasks, documentation, moderate complexity problems

### Claude Haiku 4.5
- **Best for**: Quick fixes, simple tasks, rapid prototyping
- **Strengths**: Fast response, lightweight, cost-effective
- **Use Case**: Small-scale modifications, testing, batch operations

## Gemini (Google)

### Gemini 3 Pro (High)
- **Best for**: Repository-wide analysis, deep research, complex designs
- **Strengths**: High precision, hyper-large context, multi-modal capabilities
- **Use Case**: Comprehensive codebase review, advanced research tasks

### Gemini 3 Flash
- **Best for**: High-speed coding, documentation, quick fixes
- **Strengths**: Rapid response, good quality-to-speed ratio
- **Use Case**: Batch processing, quick investigations, documentation generation

## Open Source Models (Ollama)

### qwen2.5:7b / llama3.1:8b
- **Best for**: Local execution, confidential experiments, offline work
- **Strengths**: Privacy-preserving, customizable, no quota limits
- **Use Case**: Security-sensitive prototyping, training data processing

## Selection Guide

| Task Type | Recommended Model | Reason |
|-----------|------------------|--------|
| Large-scale refactoring | Claude Opus 4.5 | Deep reasoning required |
| New feature implementation | Claude Sonnet 4.5 | Balanced capability & efficiency |
| Quick bug fix | Claude Haiku 4.5 | Speed-optimized |
| Complex research | Gemini 3 Pro | Large context & analysis |
| Fast documentation | Gemini 3 Flash | Speed-optimized quality |
| Confidential work | Ollama (qwen/llama) | Privacy-preserving |
