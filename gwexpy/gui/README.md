## 概要 / Overview

`pyaggui` は diaggui ライクなGUIツールで、gwexpy を使った時系列データの可視化と解析を行います。

## 起動方法 / How to Run

```bash
cd gwexpy/gui
python pyaggui.py
```

## 依存関係 / Dependencies

- PyQt5
- pyqtgraph
- gwexpy

## 既知の問題 / Known Issues
